# Security Policy

AI Capacity Optimizer is currently an MVP and should be treated as local demo software.

## Supported Versions

Only the latest `main` branch is supported during the alpha stage.

## Reporting a Vulnerability

Open a private security advisory or contact the repository owner before publicly disclosing issues that involve:

- credential leakage
- unsafe proxying of real provider API keys
- account or billing data exposure
- request routing bugs that could send data to the wrong provider

## Data Handling

The default repository data is mock/demo data. Do not commit real usage logs, customer prompts, provider API keys, or billing exports.
